# vue-go-blog
本博客项目基于vue+typescript以及gin框架搭建<br>
本意是验证自己前后端能力。<br>
本博客会用于记录我的日常生活，同时也作为我的学习笔记。


# backend
    cd backend
    export GOPROXY=https://mirrors.aliyun.com/goproxy/
    export GO111MODULE=on
    go run main.go

# frontend
    cd frontend
    npm install
    npm run serve
